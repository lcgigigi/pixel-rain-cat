───────────────────────────────────────────────────────
🚀 快速开始（3步即可运行）
───────────────────────────────────────────────────────

方法1：使用脚本启动（推荐）
   双击运行 "启动项目.bat" 即可

方法2：手动启动
   1. 打开终端（PowerShell 或 CMD）
   2. 运行：npm install
   3. 运行：npm run dev

启动后浏览器会自动打开 http://localhost:3000

───────────────────────────────────────────────────────
📱 在手机上查看
───────────────────────────────────────────────────────

开发阶段 - 局域网访问：
  1. 确保手机和电脑在同一WiFi
  2. 查看终端显示的 Network 地址
  3. 在手机浏览器输入该地址

正式使用 - 部署到网络：
  推荐使用 Vercel（完全免费，3分钟部署）
  详见：DEPLOYMENT.md

【推荐】Vercel - 最简单（免费）
  • 访问 vercel.com
  • 连接 GitHub 导入项目
  • 点击部署

Netlify - 同样简单（免费）
  • 访问 netlify.com
  • 导入项目
  • 自动部署

Cloudflare Pages - 国内快（免费）
  • 访问 pages.cloudflare.com
  • 部署项目

GitHub Pages - 需配置（免费）
  • 按照 DEPLOYMENT.md 配置

云服务器 - 完全控制（付费）
  • 构建后上传 dist 文件夹
  • 配置 Nginx

详细教程请看：DEPLOYMENT.md

❓ 常见问题
───────────────────────────────────────────────────────

Q: 第一次使用怎么启动？
A: 双击 "启动项目.bat" 或运行 npm install 然后 npm run dev

Q: 如何在手机上查看？
A: 开发时用局域网访问，正式用推荐部署到 Vercel

Q: 如何修改页面内容？
A: 编辑 src/views/Home.vue 或 About.vue 文件

Q: 如何添加新页面？
A: 在 src/views/ 创建新 .vue 文件，然后在路由配置

Q: 部署用哪个平台？
A: 推荐 Vercel，完全免费，3分钟搞定

Q: 需要 UniApp 吗？
A: 不需要！这是标准 Vue3 项目，已做移动端优化
